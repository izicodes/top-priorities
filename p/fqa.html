FAQs

How to add the widget to my Notion page?

  1. In an empty space, type /embed and press enter
  2. Paste the copied link of the widget
  3. Click the Embed link button - that's it!

I added the widget but it's not loading?

I'm super sorry about that! If you are trying to embed/view the widget on the Notion mobile app, unfortunately Notion will not load the widget.

However, on the Notion web app and desktop app, the widget should work as normal, exactly as you can see in the preview version of your widget.

(1) Note: When you do embed the widget through the web/desktop app but can't see the widget in the mobile app, the widget does not disappear nor does your listed tasks in the widget when you return back to the web/desktop app.

(2) Note: I am working to make the mobile version to be functional as well!

I tried adding another widget onto my page but it's not working?

  • Make sure each of the widgets has a unique name give. You can do this via the "Give it a name (optional)"
